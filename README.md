# NWG I – Übersicht & Übungsaufgaben

Statische Website für Naturwissenschaftliche Grundlagen I: Themenübersicht (`index.html`) mit Fortschritts-Tracking (Cookies) und Übungsaufgaben mit Hinweisen und Lösungswegen.

## Deployment

```bash
git clone <this-repo-url>
```

Dieses Repository enthält **nicht** die Vorlesungsfolien (`Vorlesungen/`), da es sich dabei um urheberrechtlich geschütztes Material des Dozenten handelt. Die `Vorlesungen/`-Ordnerstruktur muss separat auf den Server kopiert werden (z. B. per `scp`/`rsync`), damit die entsprechenden Links in `index.html` funktionieren.
